# מבחן Cross-Reference למודל שפה

זהו סט נתונים סינתטי בעברית לבדיקת יכולת של מודל שפה לבצע הצלבת מידע.

המטרה היא לבדוק טענה צרה ומעשית:

- המודל לא מקבל זרמי מידע גולמיים בנפח גבוה.
- המודל מקבל חבילת ראיות מצומצמת שכבר עברה סינון ועיבוד ראשוני.
- עליו לחבר בין ישויות, כינויים, זמן, מיקום, דפוסי עבר והסברים חלופיים.
- עליו להבדיל בין ראיה ישירה, אינדיקציה עקיפה והשערה.

## הערת בטיחות והקשר

שמות המקומות הם שמות אמיתיים ומוכרים בישראל כדי להקל על ההבנה המרחבית.

כל החברות, האנשים, האירועים והקשרים בסיפור הם בדיוניים לחלוטין. אין כאן טענה או רמיזה על פעילות אמיתית של אדם, חברה, ארגון או מקום.

## קבצים

- `entity_registry_he.csv` - רשימת ישויות, כינויים וקשרים.
- `locations_he.csv` - רשימת מקומות אמיתיים/מוכרים עם תיאור תפקיד בסיפור הבדיוני.
- `events_he.csv` - אירועים מובנים לאחר עיבוד ראשוני.
- `reports_he.md` - דיווחים חופשיים והערות אנליסט.
- `model_prompt_he.md` - הנחיה למודל.
- `answer_key_he.md` - תשובת ייחוס וסרגל ניקוד.
