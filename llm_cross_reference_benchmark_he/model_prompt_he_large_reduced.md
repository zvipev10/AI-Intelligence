# הנחיה למודל - חבילת מועמדים מצומצמת

קיבלת קבצי רקע בעברית ואת `candidate_package_he.md`, שהוא פלט של שכבת סינון שאינה LLM מתוך זרם אירועים גדול.

המשימה: הערך את חבילת המועמדים והסק האם קיימת השערת cross-reference משמעותית.

השתמש גם ב:

- `entity_registry_he.csv`
- `locations_he.csv`
- `reports_he.md`

ענה בעברית בלבד.

פורמט:

- שורה תחתונה בפסקה אחת.
- ראיות תומכות עם מזהי אירוע.
- כינויים/זהויות חשובים.
- מסיחים והסברים חלופיים.
- רמת ביטחון.
- בדיקות המשך.

אל תטען לוודאות. הבדל בין ראיה ישירה לבין הסקה.
